Text Command Palette Test

Select text in the textarea below and press Ctrl+K (or Cmd+K on Mac) to open the command palette.

Test Cases

Uppercase: "hello world" → "HELLO WORLD"
Title Case: "hello world example" → "Hello World Example"
Camel Case: "hello world example" → "helloWorldExample"
Kebab Case: "Hello World Example" → "hello-world-example"
Reverse: "hello" → "olleh"
Trim: " hello world " → "hello world"
Remove Punctuation: "Hello, world!" → "Hello world"
Count Words: "Hello world example" → "Words: 3, Characters: 18"
Format Date to IST: "1704067200000" → "1/1/2024, 5:30:00 AM"
